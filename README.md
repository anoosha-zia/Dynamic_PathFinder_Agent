\# Dynamic Pathfinding Agent



\## Project Overview

This project implements a Dynamic Pathfinding Agent using:



\- A\* Search

\- Greedy Best First Search

\- Real-time obstacle spawning

\- Dynamic re-planning

\- Pygame GUI visualization



\## Features Completed

\- Interactive Grid

\- Random Maze Generation

\- Fixed Start and Goal Nodes

\- Manual Obstacle Placement



\## Technologies Used

\- Python

\- Pygame

\- Heapq

\- Object-Oriented Design



\## Controls

\- R → Generate Random Maze

\- C → Clear Grid

\- Left Click → Add Wall

\- Right Click → Remove Wall
